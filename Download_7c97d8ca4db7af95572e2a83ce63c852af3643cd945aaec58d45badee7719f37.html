Runemod pre-alpha

How to try the Runemod pre-alpha:

1. Install jdk 11. You may get the installer here. (If you already have jdk 11 you may skip this)

2. Download the runemod launcher here and run it.

3. If you are not using a jagex account, you can go ahead and login.
However, if you are using a jagex account, you need to follow these steps to login:
Setting up a jagex account in runemod alpha - Google Docs
